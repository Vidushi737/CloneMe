# CloneMe: AI-Powered Digital Twin Platform

CloneMe is an AI-powered Digital Twin platform. The application learns from a user's journals, goals, habits, achievements, reflections, and notes to construct a digital representation of their personality, growth, and behavior over time.

## Project Structure

```
CloneMe/
├── frontend/             # React + Vite + TS + Tailwind CSS
├── backend/              # FastAPI + SQLAlchemy + SQLite/Postgres
├── docs/                 # Documentation (Installation, API, Dev guides)
├── docker/               # Dockerfiles for services
├── scripts/              # Helper scripts
├── README.md             # Project introduction
├── .env.example          # Sample environment variables
└── docker-compose.yml    # Docker services orchestration
```

## Core Features

- **Store Memories & Journals**: Write daily logs and upload notes.
- **Track Growth**: Real-time habit tracking and goal management.
- **Analyze Behavior**: Mood tracking and analytics visualizations.
- **AI Twin Chat**: RAG-based chatbot trained on your own memories.
- **Personality Evolution Engine**: Personality report, strengths, weaknesses, and growth indicators.
- **Life Timeline**: Timeline organized automatically into life chapters.
- **Future Self Predictor**: Projections based on goals and habit completion consistency.
